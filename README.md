# Final-year-project_Developing Probabilistic Weather Prediction Model
This project is an undergraduate graduation project that focuses on the field of time series prediction. It integrates classic time series models with Bayesian probability prediction models, completes data fitting and future trend prediction, and builds a set of web visualization dashboards. Displaying model prediction results through intuitive charts, achieving model comparison, result visualization, and interactive browsing, suitable for scenarios such as temperature/time series data prediction.
# Core functionality
Classic time series model
Implement the training, prediction, and comparative analysis of AR, ARIMA, and VAR models, demonstrating the fitting effect and prediction accuracy of the models.

Bayesian prediction model
Build Bayesian AR and Bayesian VAR probability prediction models to output uncertainty prediction, trend inference, and extreme value analysis.

# Multi dimensional visualization chart
Includes test set fitting graph, future prediction graph, fan-shaped prediction graph, time-series spaghetti graph, and extreme event heatmap.

# Tech stack
Prediction models: AR/ARIMA/VAR/Bayesian AR/Bayesian VAR
Front end display: HTML5+CSS3 (DASHBOARD)
Visualization: Python data visualization (chart generation)
Deployment: GitHub Pages (online access)
Operating environment: All modern browsers (Edge/Chrome recommended)
